# ExamsQuiz
Create a random quiz in latex and ASCII format
## How to install
Open a terminal and type:

`git clone https://github.com/dodogabrie/ExamsQuiz.git`

## How to use

ExamsQuiz
|
+-- _esame
|
+-- _domande
    +-- _1_CapName
    |   +-- Quest1.tex
    |	+-- Quest2.tex
    +-- _2_CapName
	+-- Quest1.tex
	+-- Quest2.tex
